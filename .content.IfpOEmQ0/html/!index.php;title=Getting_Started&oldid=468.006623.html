Introduction

Hello new Battle Dawner! If you wish to try out the game before registering, you can simply go here and click "Play now!" on the right hand side. The account created for this is temporary, and allows you to "Try before you buy" (so to speak). You will then be put through the same tutorial that you would were you on a new account: You will pick a server (From Earth, Mars and Fantasy) and then place, name and build your colony up in a series of easy-to-understand steps.


Types of Server

There are three "Settings" or maps for Battle Dawn. Firstly, Earth, which is a large earth-based map. Secondly, there is Mars, which is a red planet themed map and finally, Fantasy, which is based off a medieval sword-and-magic style setting. Beyond the layout and size, there is very little mechanical difference between the maps currently.


The Rules and Rule Enforcement

The rules to the game can be found here. Rule enforcement is done in-game by a designated World Administrator, a list of which for each server can be found here. You can send a message to the administrator of a world by clicking the "A" button in the top right hand corner and clicking on his colony. The administrators try to reply to all messages within 24 hours, but if you do have to wait longer than that, do not send another message.


Help me!

There are many places you can go for help in Battle Dawn. The first place is here, at this very Wiki! but particularly here. Secondly, we have an Adopt a Newb mentoring scheme. Simply sign up following the instructions, and you will have an experienced mentor assigned to you for your help needs. Additionally, there is the Newbie Questions sub forum on the Battle Dawn forums. The people there are generally very helpful! We were all new to the game once, so we know that it can be a bit overwhelming.


The Forums

The Forums are a very frequently visited part of Battle Dawn, the community is very active. In the forum, you can discuss battles and wars, ask for help, have a debate or relax a bit in the spam section. The forums are a hotbed of information: whether it be on the latest upcoming alliance, game updates or announcements on things like downtime.

The forum is accessed using the same user name and password as your game account, be warned though, that offences on the forum can (and have in the past) led to in-game sanctions, so remember, follow The Rules at all times on the site.
